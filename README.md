# 三术 (Sanshu) — Fork 二开记录

基于 [yuaotian/sanshu](https://github.com/yuaotian/sanshu) 的二次开发分支。

## 二开内容

### 弹窗交互增强
- 本地提示词增强功能，移除对外部增强服务的依赖
- 结构化响应格式：用户输入、选项、图片、文件引用、条件偏好分区传递给 AI
- 图片 Badge 支持索引序号和文件名显示
- 截图粘贴自动生成带时间戳的唯一文件名
- 拖拽文件/图片时自动聚焦窗口和编辑器
- Markdown 渲染对齐 Naive UI 样式 + 代码块复制按钮修复

### UI/UX 重构
- Fork 品牌重塑与桌面 UI 刷新
- UI 标准化：主题色、间距、排版统一
- 置顶按钮常驻标题栏
- 本地字体加载，消除启动闪烁
- 生产环境构建样式修复
- Toast/Modal 不再被标题栏遮盖
- 关闭按钮双色反馈：MCP 弹窗首次关闭黄色警告，再次关闭红色确认

### MCP 工具链优化
- `zhi` 子代理调用约束
- `zhi` 提示词抗上下文压缩 + 强制 `project_root_path`
- 执行偏好拆为独立 Content::text，修复与用户输入粘连
- 连续取消关闭保护：两次取消自动结束对话，防止无限弹窗循环
- 统一弹窗协议 + WebView2 多实例黑屏修复

### 自定义提示词系统
- 自定义提示词独立存储（不依赖主配置文件）
- 条件性提示词：开关控制、MCP 工具联动
- 提示词工具 ID 一致性修复

### 自动更新
- 应用启动时自动检查更新
- 发现新版本 → 确认下载 → 进度展示 → 确认重启
- GitHub Release 自动发布（auto-tag workflow）

### 工程化
- 解决全部 48 个 cargo clippy 警告
- 移除遗留兼容代码和死代码
- Vite 构建产物 chunk 命名简化
- TypeScript 6.0 + Vue TS 插件
- 原子化配置文件读写
- CI 简化 + 版本管理脚本（`pnpm bump`）
- 图标工坊弹窗布局优化
